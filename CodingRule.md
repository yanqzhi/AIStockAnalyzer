# Coding Rule

## Python

- Python 3.12+
- Type Hint Required
- Google Docstring
- PEP8
- Ruff

## Project

- Every Commit Must Run
- No Hard Code
- Config First
- Unit Test Required
- Logging Required

## Git

- Feature Branch
- Small Commit
- Meaningful Commit Message

## Principle

Readable > Clever

Simple > Complex

Maintainable > Fast
